# BridgeEd

## Brand Identity & Design System (v1)

---

# 1. Brand Positioning

## Brand Mission

BridgeEd exists to **bridge foundational learning gaps** in literacy and numeracy through clear diagnostics, actionable remediation, and measurable progress tracking.

## Brand Personality

* Trustworthy
* Structured
* Supportive
* Evidence-driven
* Practical

BridgeEd should feel calm, reliable, and professional — suitable for schools, districts, and national education bodies.

---

# 2. Visual Identity System

## 2.1 Primary Color — Bridge Blue

Represents trust, institutional credibility, and stability.

Hex:

```
#1E3A8A
```

Usage:

* Primary buttons
* Active navigation states
* Headers
* Key highlights

---

## 2.2 Secondary Color — Progress Green

Represents growth, mastery, and remediation progress.

Hex:

```
#15803D
```

Usage:

* Success states
* Mastery indicators
* Progress bars
* Positive analytics signals

---

## 2.3 Accent Color — Insight Gold

# 3. Typography System

## Font Style

Use clean, highly readable sans-serif fonts.

Recommended:

* Inter
* Manrope
* System UI fallback

## Hierarchy

* H1: Bold, strong page headers
* H2: Section headers
* H3: Card headers
* Body: Regular weight
* Caption: Smaller, muted text

All typography must prioritize readability on mobile screens.

---

# 4. UI Principles

## 4.1 Mobile-First

* Large tap targets
* Clear spacing
* Simple navigation hierarchy
* Minimal cognitive overload

## 4.2 No Visual Noise

* No random gradients
* No flashy AI neon styling
* No inconsistent shadows

Use subtle shadows only where needed for card elevation.

## 4.3 Clear Data Communication

* Use green for mastery and improvement
* Use red for critical gaps
* Use neutral tones for informational states
* Avoid excessive color combinations

---

# 5. Component Design Rules (Mantine + Tailwind)

* Mantine components must use theme tokens.
* Tailwind utilities allowed for layout and spacing only.
* Border radius should be consistent (medium rounded).
* Buttons:

  * Primary: Bridge Blue background
  * Secondary: Outline with Bridge Blue border
  * Success: Progress Green

---

# 6. Iconography

* Use simple line icons.
* Avoid default AI robot icons.
* Icons must reinforce clarity, not decoration.

---

# 7. Tone of Voice

BridgeEd communication style should be:

* Clear
* Direct
* Encouraging
* Professional

Avoid technical jargon in teacher-facing areas.
Avoid exaggerated marketing language.

---
